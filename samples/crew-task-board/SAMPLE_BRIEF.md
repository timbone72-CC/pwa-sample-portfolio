# Crew Task Board Sample Brief

## 1. Sample ID

crew-task-board

## 2. Purpose

Crew Task Board is a lightweight PWA sample for crew leads, foremen, and small team managers.

It helps assign and track daily crew tasks without needing a backend, login, or shared account system.

## 3. Primary User

Crew lead, foreman, or small team manager.

## 4. Main Workflow

Create a task, assign it to a crew member, set priority, update status, and keep short notes.

## 5. Minimal Features

- Add task
- Assign task to crew member
- Set status: Not Started / In Progress / Done
- Set priority
- Add notes
- Save locally in the browser
- Mobile-friendly layout
- Installable PWA
- Unique app icon/logo

## 6. Standalone Use

Yes.

This app must be useful by itself.

## 7. App Relationship Note

Crew Task Board can stand alone, but it may also connect to:

- Job Tracker — assigned tasks become completed job records
- Equipment Check Log — tasks can include equipment issues
- Service Call Log — service calls can generate crew tasks

Connection status:

Optional / future.

## 8. Boundaries

No backend.

No login.

No payments.

No live app-to-app connection yet.

No FieldLedger changes.
